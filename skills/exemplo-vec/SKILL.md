# exemplo-vec

Skill mínima usada para confirmar que a distribuição de skills da VEC está operando ponta a ponta.

## Quando usar
- O usuário pede para testar/validar o SkillHub ou o marketplace da VEC.
- Quer confirmar que o auto-update trouxe a versão nova.

## Passos
1. Confirme que a skill foi carregada, informando o nome `exemplo-vec` e a versão.
2. Responda: "SkillHub operando — skill exemplo-vec carregada com sucesso."
